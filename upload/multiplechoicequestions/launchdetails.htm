To launch your presentation:

Multiple Choice Questions

The link above will load the presentation into a new, correctly sized window, with toolbars, statusbars and scrollbars removed. The window will be centered on screen.

Below, you'll see how to incorporate the same code as used above to launch a lesson from your own web page.

Instructions for a Quick Launch for Users:

Upload the contents of the directory you exported this file to a directory on your server, and direct users to the index.htm page in that directory. It provides a link that will launch the lesson in the correctly sized browser window.

Files in the Export Directory:

  • launchfull.htm is the actual data file. This is the file that would be launched by a Learning Management System for example, and the file launched by the first link above.
  • multiple choice questions.html, borders.htm, scatter.htm, and multiple choice questionsfs.htm are files required by the lesson or presentation, but are not run directly.
  • launchdetails.htm is this file, containing instructions on how the launch the lesson from your web page.
  • index.htm is a file that you can direct users to that will allow them to launch the lesson in its own browser window, at the correct size.
  • report.htm is a report that contains details about the exported file size.
  • Any other htm files are support files for the main data file.

Adding this link to your own web page:

Copy and paste the following text into your web page. It can be used to launch your presentation into the correctly sized window.

<A HREF="javascript:winname='KP' + Math.round(Math.random() * 1024) +1;tl = (screen.width - 932) / 2;tt = (screen.height - 660) / 5;KP=window.open('launchfull.htm', winname, 'left=' + tl + ',top=' + tt + ',status=no,resizable=yes,toolbar=no,hotkeys=no,scrollbars=no,width=932,height=660');if (KP){KP.focus()}else{alert ('You may have a popup blocker running. Please disable this from this site before this link will work.')}" onMouseOver="window.status='Multiple Choice Questions';return true" onMouseOut="window.status='';return true">Multiple Choice Questions</A>
